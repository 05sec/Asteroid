## 开始使用
```bash
git clone https://github.com/wuhan005/Asteroid
```
Unity 2019.1.0f2 打开项目，点击`Main Camera`摄像机，在`Inspector`面板的`Main Controller`中设置`Url`为后端 Websocket 接口，播放/编译打包即可。
